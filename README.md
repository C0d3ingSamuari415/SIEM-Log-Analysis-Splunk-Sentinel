# SIEM Log Analysis (Splunk & Azure Sentinel)

This project demonstrates hands-on SIEM analysis using Splunk and Azure Sentinel. It includes sample logs, detection queries, and investigation reports aligned with SOC workflows and MITRE ATT&CK.

## Contents
- Sample authentication logs
- Splunk detection queries
- Azure Sentinel KQL queries
- Investigation reports
- MITRE ATT&CK mapping

## Skills Demonstrated
- Log analysis
- Threat detection
- Query writing (SPL & KQL)
- Incident documentation
- Root-cause analysis

## Folder Structure
- /logs — sample log files
- /splunk_queries — SPL queries
- /sentinel_kql — KQL queries
- /findings — investigation reports
